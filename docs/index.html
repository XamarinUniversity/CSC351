Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin.Forms development environment with either a simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

In this class you will learn how to write multithreaded applications to keep your applications' UI responsive and take advantage of the performance benefits of multicore processors.

Objectives

  1. Using multiple threads in mobile apps
  2. Creating threads and tasks
  3. Dealing with exceptions
  4. Canceling tasks

Lecture

CSC351 - Writing multi-threaded mobile applications.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Explore an unresponsive app
    Exercise
    In this exercise, we will examine an application with an unresponsive UI and explore the specific causes.
  2. Partition work over multiple threads
    Exercise
    In this exercise, we will partition our CPU bound work to take advantage of multithreading and multicore processors.
  3. Handle Exceptions that occur in Tasks
    Exercise
    In this exercise, we will throw exceptions from within a Task and manage it in a try/catch block.
  4. Use cancellation tokens to stop long running tasks
    Exercise
    In this exercise, we will use cancellation tokens to interrupt long running tasks.
